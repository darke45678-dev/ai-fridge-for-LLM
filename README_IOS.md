# 🚀 Kitchen AI - iOS 實體 App 安裝指南 (免 App Store)

本指南幫助您將 **Kitchen AI** 繞過 Apple App Store，直接以「實體安裝檔 (.ipa)」的方式載入您的 iPhone。

---

## 🛠️ 第一階段：從 GitHub 下載編譯好的 App

我們已在 GitHub 倉庫中配置了 **GitHub Actions (雲端 Mac 編譯工廠)**。只要推送程式碼，GitHub 就會自動幫您打包 iOS 安裝檔。

1. **前往 GitHub 倉庫的 Actions 頁面**
   👉 [https://github.com/darke45678-dev/ios-fridge-test/actions](https://github.com/darke45678-dev/ios-fridge-test/actions)

2. **點擊最近一次成功的 `Build iOS IPA (Standalone)` 任務** (綠色勾勾 ✅)

3. **下載編譯產物**：
   - 捲動到頁面最下方的 **「Artifacts」** 區域
   - 點擊 **`KitchenAI-iOS-Standalone-IPA`** 下載（約 37 MB）
   - 解壓縮 `.zip` 後，裡面就是 **`KitchenAI.ipa`**

---

## 📲 第二階段：用 Sideloadly 安裝到 iPhone (Windows)

因為 iPhone 只允許安裝來自 App Store 的軟體，我們需要用 **Sideloadly** 進行「個人簽名」。

### 準備工作
1. 下載並安裝 **Sideloadly**：[https://sideloadly.io](https://sideloadly.io)
2. 用傳輸線將 iPhone 連接到電腦
3. 手機出現「**信任此電腦？**」→ 點擊 **「信任」**

### 安裝步驟
1. 打開 **Sideloadly**
2. 將 `KitchenAI.ipa` **拖進** Sideloadly 視窗
3. 在 **「Apple ID」** 欄位輸入您的 Apple 信箱
4. 點擊 **「Start」**，輸入您的 Apple 密碼（由 Apple 官方驗證，安全）
5. 等待安裝完成 ✅

### 信任開發者（iPhone 端最後一步）
安裝完後，App 桌面會出現，但第一次開啟會顯示「不受信任的開發者」：

> **iPhone 設定 → 一般 → VPN 與裝置管理 → 找到您的 Apple ID → 點「信任」**

## 🔄 如何更新 App？

每次您修改了程式碼，只需在終端機輸入：

```powershell
git add .
git commit -m "你的修改說明"
git push origin main
```

GitHub 會自動重新編譯。等完成後，重複「第一階段」下載新的 `.ipa`，再用 Sideloadly 覆蓋安裝即可（**不會清除 App 內的庫存資料**）。
